## 📂 Repository Structural Blueprint

The workspace is split systematically into specific enterprise modules to maximize modularity:

```text
├── 📦 Backend-Spring-Core/           # Core Spring Inversion of Control & DI Modules

    ├── src/components/                # Presentational & Container Components
